# Modifications

This derivative adds Windows 11 x64 packaging, portable path handling,
packaged runtime assets, cross-platform control discovery, and Windows CI to
the upstream Motion Previs Studio 4.1.0 baseline frozen at commit
`c257de2ad5aba1ba49b0a0a4f2a44d6b6f660288`. Windows support contributed and
maintained by **Gumbii Digital** ([github.com/GumbiiDigital](https://github.com/GumbiiDigital)).
